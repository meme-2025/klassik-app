Available Endpoints

Blocks

GET /blocks/latest

Get latest blocks with pagination

Parameters
  • limit - Number of blocks (default: 10, max: 100)
  • offset - Pagination offset
Example Request
curl https://api.kaspascan.io/v1/blocks/latest?limit=10
GET /blocks/:blockHash

Get block details by hash or number

Example Response
{
  "blockHash": "0x1a2b3c...",
  "blockNumber": 45892347,
  "timestamp": 1732867200,
  "transactions": 127,

  "reward": "250.5 KAS"
}
