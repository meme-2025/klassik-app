# Backend — Klassik

Dieses Verzeichnis enthält ein minimales Express‑Skeleton.

Schnellstart:
1. `cd backend && npm install`
2. `npm run dev` (benötigt nodemon) oder `npm start`

Endpunkte:
- GET /health
- POST /api/orders

Die statischen Frontend‑Dateien werden vom Backend unter `../frontend` ausgeliefert.

Nächste Schritte:

- Auth via Wallet‑Signaturen
